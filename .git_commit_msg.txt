feat(notes): 笔记全文搜索 + Cmd+Shift+F 快捷键

Tier 三 #3: 笔记面板加搜索 input
- input 时实时过滤 quote + text (80ms debounce)
- 匹配高亮黄色 mark
- 无结果时显示 "没有包含 X 的笔记"
- Cmd+Shift+F 焦点到搜索 (并自动打开笔记 panel)
- 按 X 删笔记 (existing) 不受搜索影响

实现:
- .notes-search input 在 .notes-panel-header 下面
- renderNotesList() 接受 query, 过滤 + highlight
- per-char escape 函数 (避 Python source 里 \\ 双重转义坑)

UX 价值: 175 章散落笔记, 用户搜 "agent" 一秒定位, 比翻页 10x.
