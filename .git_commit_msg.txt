feat(today): 顶部一屏 今日推荐 + 最近书签

UX #5: Overview hero 区下加 today-panel, 2 列布局

**今日推荐** (3 章, 跨书 ≤2 章/书):
- 在读 (0 < p < 0.9) 优先于未读 (p=0), 跳过已读 (p≥0.9)
- 仅在用户有活动 (lastRead / completed / 任一 p>0) 时才推荐, 全新用户不打扰
- 1 数字 badge + 标题 + 系列名 + 进度百分比 (在读时)

**最近书签** (≤3 条):
- 拍平 localStorage.bookmarks (按 chapterId 嵌套), 按 ts desc 排序
- 显示书签文字 (截 40 字符) 或 fallback 章节标题
- 元信息: 系列名 + 相对时间 (formatRelativeTime)

**布局**:
- desktop: 1fr 1fr grid, gap 16px, max-width 960px
- mobile (≤700px): 1 列堆叠
- 整体在 hero 与 我的阅读 看板之间, 1 屏可见

**边界 case 3 个**:
- 完全无数据: 整个 panel display:none (carousel 也 hide, 不显示空 panel)
- 只有书签无 progress: 1 列 (最近书签)
- 只有 progress 无书签: 1 列 (今日推荐)
- 都有: 2 列, 5 项
