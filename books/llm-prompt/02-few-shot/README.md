# Few-shot 与示例

> 给 LLM 看几个例子，比讲 100 条规则更管用。

## 什么是 Few-shot？

提供几个"输入 → 输出"的样例，让 LLM 模仿你的风格。

```
将以下评论分类为正面 / 负面 / 中性：

评论：今天天气真好！
分类：正面

评论：还行吧，一般般。
分类：中性

评论：太差劲了，浪费钱。
分类：负面

评论：速度快，客服态度好。
分类：正面
```

## 几个原则

1. **多样性**：覆盖各种 case
2. **边界 case**：包含容易混淆的例子
3. **格式一致**：所有样例用同一种格式
4. **3-5 个最佳**：太多反而稀释
