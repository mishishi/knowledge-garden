# 04. Newsletter 经济：2026 年最值得做的内容生意

Newsletter（邮件订阅）在 2020 年还只是小众爱好，2026 年已经是 **年增长 30%+ 的内容创业赛道**。Substack / Beehiiv / Kit 上 50 万+ 付费创作者，年收入 $40 亿+。

这一章讲清楚：Newsletter 为什么火、怎么开始、能赚多少、未来怎么走。

## Newsletter 为什么火

**原因 1：用户主权回归**

过去 10 年是平台中心化（微信公众号 / 头条 / Twitter / TikTok），算法决定生死，创作者被平台绑架。

Newsletter 是**创作者中心化**：邮件列表是创作者的资产，平台死了用户还在。

**原因 2：订阅经济成熟**

用户越来越愿意为优质内容付费。Spotify 1 亿+ 付费用户、Netflix 2 亿+、Substack 50 万+ 付费订阅创作者——订阅经济全面到来。

**原因 3：AI 降低门槛**

写 Newsletter 的执行成本从"1 周 1 篇"变成"1 天 1 篇"。AI 辅助让长尾创作者暴增。

**原因 4：广告疲劳**

用户被广告 / 算法推送烦了，主动订阅"我信任的人"。

## Newsletter 核心数据

**全球 Newsletter 市场**（2026）：

- 创作者总数：500 万+（免费 + 付费）
- 付费订阅创作者：50 万+
- 总付费订阅：3000 万+
- 年总收入：$40 亿+
- 头部 1% 创作者拿走 60% 收入

**典型 Newsletter 收入区间**：

| 订阅数 | 月收入 | 典型案例 |
|-------|--------|---------|
| < 1000 | $0 | 大部分新手 |
| 1000-10000 | $50-1000 | 长尾创作者 |
| 10000-50000 | $1000-10000 | 中部创作者 |
| 50000-200000 | $10000-50000 | 头部 |
| 200000+ | $50000-500000+ | 超级头部 |

**单位经济**：1 个付费订阅 = $5-10/月。1000 个付费订阅 = $5000-10000/月。

**订阅转化率**：免费到付费转化率平均 2-5%。1 万免费订阅 → 200-500 付费 → $1000-5000/月。

## Newsletter vs 其他内容形式

### vs 公众号

**Newsletter 优势**：全球可达、不被算法绑架、数据归自己、跨境支付。

**Newsletter 劣势**：中国受众习惯公众号，转化率低。

### vs YouTube

**YouTube 优势**：被动发现、视觉冲击、广告变现。

**Newsletter 优势**：读者注意力更集中（主动订阅 vs 被动刷）、文字深度高、变现效率高。

### vs 博客

**Newsletter 优势**：直接触达、订阅机制、推送机制。

**Newsletter 劣势**：博客有 SEO 长尾流量。

### vs 播客

**播客 优势**：陪伴感强、消费场景多（开车 / 运动 / 做饭）。

**Newsletter 优势**：消费更快、可搜索、可深度。

## Newsletter 的 7 个赛道

**1. 行业分析 / 商业洞察**

代表：Stratechery、The Information、Benedict Evans。

**收入**：头部 $50 万-200 万/年。

**门槛**：需要专业背景 + 深度思考。

**AI 替代难度**：低（核心是个人视角）。

**2. 科技 / AI 评论**

代表：Import AI、The Rundown AI、Bens Bites。

**收入**：中部 $1-10 万/年。

**门槛**：AI 速度快、能持续跟进。

**AI 替代难度**：中（AI 也能写，但缺第一手体验）。

**3. 金融 / 投资**

代表：The Generalist、Net Interest。

**收入**：头部 $30-100 万/年。

**门槛**：金融背景 + 投资圈人脉。

**AI 替代难度**：低（个人见解是关键）。

**4. 文化 / 生活方式**

代表：Derek Sivers、The Marginalian。

**收入**：长尾到中部。

**门槛**：个人魅力 + 写作能力。

**AI 替代难度**：低（个人故事 / 视角是核心）。

**5. 设计 / 创意**

代表：sidebar、Fonts In Use。

**收入**：长尾到中部。

**门槛**：设计眼光。

**AI 替代难度**：中（设计评论需要审美）。

**6. 编程 / 技术**

代表：Bytes、TypeScript Weekly。

**收入**：长尾到中部。

**门槛**：技术深度 + 写作能力。

**AI 替代难度**：中（AI 能写代码但写不出实战经验）。

**7. 健康 / 个人发展**

代表：Andrew Huberman、Peter Attia。

**收入**：头部 $10-50 万/年。

**门槛**：专业背景（医学 / 健身）。

**AI 替代难度**：低（专业经验 + 个人实验）。

## Newsletter 的 5 个阶段

### Stage 1：0 → 1000 订阅

**核心目标**：找到 100 个"种子用户"。

**方法**：

- 朋友 / 同事 / 同事的朋友
- Twitter / LinkedIn 发帖
- 写"免费但高质量"内容，建立信任
- 评论别人的内容（被看到）
- 行业社群分享

**时间**：3-6 个月。

**典型错误**：追求订阅数忽略质量。

### Stage 2：1000 → 10000 订阅

**核心目标**：建立稳定更新节奏 + 内容差异化。

**方法**：

- 每周稳定更新（建立用户预期）
- 内容差异化（不要做"AI 综述"——太多人做了）
- 跨平台导流（Twitter / LinkedIn / YouTube → Newsletter）
- 嘉宾推荐 / 互推
- 付费社群的免费引子

**时间**：6-18 个月。

### Stage 3：10000 → 50000 订阅

**核心目标**：商业化。

**方法**：

- 开付费订阅（转化率 2-5%）
- 定价 $5-10/月 或 $50-100/年
- 给付费用户额外价值（社群 / 直播 / Q&A）
- 找赞助（sponsor）—— 1 个赞助 $500-5000
- 卖自己的产品（课程 / 工具 / 周边）

**时间**：1-2 年。

### Stage 4：50000+ 订阅

**核心目标**：规模化。

**方法**：

- 团队化（1 人变 3-5 人）
- 多渠道（Newsletter + YouTube + Podcast + Twitter）
- 品牌化（线下活动 / 出版 / 周边）
- 多元化收入（订阅 + 广告 + 产品 + 投资）

### Stage 5：退出 / 长期经营

**核心目标**：建立长期资产。

**方法**：

- 卖给媒体公司 / 私募
- 转型做基金 / 投资
- 持续经营（很多创作者终身做）

## Newsletter 收入模型

**收入 1：付费订阅**

平均定价：$5-10/月 或 $50-100/年。

**实战定价**：

- 年轻人 / 大众话题：$5/月
- 专业人士 / 高价值话题：$10-20/月
- 高端专业 / 投资类：$50-200/月

**实战数据**：5% 订阅率 = 1000 付费订阅 = $5000-10000/月（年 $60K-120K）。

**收入 2：赞助（Sponsor）**

每个 newsletter 顶部或底部放 1-3 个赞助位。

**定价**：

- 1000 订阅：$100-300/期
- 10000 订阅：$1000-3000/期
- 100000 订阅：$5000-30000/期

**每周 1 期 newsletter**：10000 订阅 = $4000-12000/月。

**收入 3：产品**

卖自己的：

- 课程（$100-500/人）
- 工具（一次性 $50-500 或订阅 $10-50/月）
- 模板 / 资料包（$20-100）
- 1v1 咨询（$200-500/小时）
- 社群（$20-50/月）

**实战数据**：产品收入通常是订阅收入的 1-3 倍。

**收入 4：投资 / 持股**

头部创作者用 Newsletter 建立影响力后投资早期项目。

**典型案例**：Stratechery 的 Ben Thompson 通过 Newsletter 早期投资了 Shopify / Substack 等公司。

## 我自己的 Newsletter 工作流

### 我的定位

个人 Newsletter 知识库，关注 AI / Agent / 工程实践。

### 工作流

```
选题（每周日 30 分钟）：
- 浏览 RSS / Twitter / Reddit / HackerNews
- AI 总结本周热点
- 人选 5 个值得写的话题

写稿（周一-周三，每天 2 小时）：
- Claude 出初稿 + 大纲
- 人重写（保留个人视角）
- 配图（Midjourney / 即梦）

发布（周三 / 周六 30 分钟）：
- Beehiiv 发布
- 自动发推 / LinkedIn
- 评论区回复（AI 自动回复基础问题）
```

### 收入现状（参考但不公开）

每月 $200-500（小规模）。

**目标**：3 个月内做到 5000 订阅 + 200 付费 = $1000/月。

## 实战建议

**想做 Newsletter 但不知道从哪开始**：

**第一步**：选一个你**真正懂** + **持续有兴趣**的话题。不要追热点。

**第二步**：写 5 篇免费内容（不要急着收费）。验证自己能不能坚持 + 内容质量。

**第三步**：找 10 个朋友订阅。让他们反馈。

**第四步**：公开发布到 Twitter / LinkedIn / 小红书。开始积累订阅。

**第五步**：3 个月后，订阅过 100 考虑付费订阅。

**记住**：Newsletter 是长期游戏，**3 个月没收入很正常**。坚持 1-2 年的创作者才能稳定赚到钱。
