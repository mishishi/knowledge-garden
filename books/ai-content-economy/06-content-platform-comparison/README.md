# 06. 内容平台对比：Newsletter 之外的 6 大选择

Newsletter 不是唯一选择。2026 年的内容创作者分布在多个平台。这一章对比 6 大主流内容平台——Substack / Beehiiv / YouTube / TikTok / 小红书 / Medium / 个人站——帮你找到适合自己的。

## 6 大平台速览

| 平台 | 主要形态 | 受众特点 | 变现机制 | 创作者数 | 推荐场景 |
|------|---------|---------|---------|---------|---------|
| Newsletter | 长文邮件 | 高质量 / 高粘性 | 付费订阅 + 广告 | 50 万+ | 深度内容 |
| YouTube | 视频 | 全球 / 大众 | 广告 + 赞助 | 5000 万+ | 知识 / 娱乐 |
| TikTok | 短视频 | 年轻 / 大众 | 创作者基金 + 直播 | 1 亿+ | 娱乐 / 流量 |
| 小红书 | 图文 / 短视频 | 中国 / 女性 | 品牌合作 + 电商 | 1 亿+ | 种草 / 生活方式 |
| Medium | 长文 | 全球 / 知识 | 付费墙 + 会员 | 100 万+ | 写作 |
| 个人站 | 任意 | 自有 | 自定 | N/A | 品牌沉淀 |

## Newsletter

**形态**：邮件订阅的长文。

**核心优势**：

- 用户主权（创作者直接拥有读者）
- 订阅制（稳定收入）
- 全球可达
- 深度内容友好

**核心劣势**：

- 没有算法推荐（依赖主动传播）
- 用户增长慢
- 写作品质要求高

**变现**：

- 付费订阅（最主流）
- 赞助 / 广告
- 产品销售

**适合**：愿意长期写、有深度见解的创作者。

**代表**：Stratechery、The Information、Casey Newton's Platformer。

## YouTube

**形态**：视频。

**核心优势**：

- 全球最大视频平台（20 亿+ 月活）
- 算法推荐（被动发现）
- 多变现机制（广告 / 赞助 / 会员 / 商品）

**核心劣势**：

- 制作成本高
- 算法变化影响大
- 竞争激烈

**变现**：

- 广告收入（每 1000 次播放 $1-20，取决于领域）
- 频道会员（$5-10/月）
- Super Chat / Super Thanks（直播打赏）
- 品牌合作
- 商品（YouTube Merch）

**数据**：

- 100 万订阅 = $5000-30000/月广告收入
- 头部频道年广告收入 $100 万+

**适合**：愿意出镜 / 录视频、有持续创作能力。

**代表**：MKBHD、Linus Tech Tips、Veritasium。

## TikTok

**形态**：短视频（15 秒 - 10 分钟）。

**核心优势**：

- 算法推荐极强（0 粉丝也能爆）
- 年轻用户多（Gen Z）
- 增长爆发力强

**核心劣势**：

- 内容生命周期短（视频 3 天后基本消失）
- 变现效率低（除电商外）
- 平台政策变化大

**变现**：

- TikTok Creator Fund（$0.5-1 / 1000 播放）
- 直播打赏
- 品牌合作
- TikTok Shop（电商）
- 推广自己其他平台

**数据**：

- 100 万粉丝 TikTok = $1000-5000/月
- 头部 TikToker = $10K-100K/月

**适合**：能持续产出短视频、年轻人受众。

**代表**：MrBeast、Charli D'Amelio、国内各种头部。

## 小红书

**形态**：图文 + 短视频（中国市场）。

**核心优势**：

- 中国女性用户为主（消费力强）
- 品牌合作 / 电商变现效率高
- 长尾流量（搜索友好）

**核心劣势**：

- 受众偏女性（部分话题受限）
- 中国监管严格
- 跨平台兼容性差

**变现**：

- 品牌合作（1000 粉丝 = $50-300 / 帖）
- 小红书电商
- 直播带货
- 推广自己的产品 / 服务

**数据**：

- 10000 粉丝小红书 = 月收入 $500-3000
- 头部博主 = 年收入 $100K-1M+

**适合**：面向中国市场、生活方式 / 美妆 / 教育类内容。

**代表**：各领域头部博主（具体名字小红书上有大量列表）。

## Medium

**形态**：长文（类似博客）。

**核心优势**：

- 内置读者社区（喜欢阅读的人）
- Partner Program 收入分成（按阅读时长）
- 写作体验好

**核心劣势**：

- 流量分成低（$0.01-0.05 / 次阅读）
- 严格的内容审核
- 算法不如 Substack

**变现**：

- Medium Partner Program（$10-500/月大多数创作者）
- 推广自己的产品 / 服务

**数据**：

- 1000 关注 = $50-200/月
- 头部 Medium 作家 = $1000-5000/月

**适合**：纯写作、喜欢 Medium 社区、不需要高收入。

**代表**：各种行业思考者、设计师、写作者。

## 个人站（Personal Site）

**形态**：完全自己控制的网站（Ghost / WordPress / 自建）。

**核心优势**：

- 完全控制（品牌 / 数据 / 内容）
- 无平台风险
- 可以做任何事

**核心劣势**：

- 需要技术能力（或花钱请人）
- 没有内置流量（要自己推广）
- SEO 是关键

**变现**：

- 自定（付费墙 / 会员 / 产品销售 / 广告）

**工具栈**：

- **Ghost**（开源 Newsletter 平台）—— 最常用
- **WordPress + Substack 插件**
- **Hugo / Astro**（静态站生成）
- **Notion + Super.so**（零代码）

**实战成本**：$30-100/月（域名 + 托管 + 邮件）。

**适合**：技术能力强 / 重视品牌沉淀 / 长期主义。

**代表**：无数个人站（derek sivers / patio11 / swyx 等都是个人站）。

## 6 平台横评

### 增长难度

```
TikTok < 小红书 < YouTube < Medium < Newsletter < 个人站
```

TikTok 算法最强，0 粉丝也能爆。个人站最难，要完全自己引流。

### 变现效率

```
Newsletter > YouTube > 小红书 > TikTok > 个人站 > Medium
```

Newsletter 单位读者变现最高（直接付费）。Medium 分成最少。

### 内容生命周期

```
Newsletter = 个人站 > Medium > YouTube > 小红书 > TikTok
```

Newsletter 邮件永远在用户邮箱里。TikTok 视频 3 天后基本没人看。

### 创作门槛

```
TikTok < 小红书 < YouTube < Newsletter < Medium < 个人站
```

短视频最易上手。个人站最难。

### 受众粘性

```
Newsletter > 个人站 > Medium > YouTube > 小红书 > TikTok
```

邮件订阅者最忠诚（主动订阅）。TikTok 用户最"无忠诚"。

## 实战策略：1+N 模型

**核心**：1 个主平台（深耕）+ N 个辅平台（分发）。

**典型组合**：

**策略 A：Newsletter + Twitter / LinkedIn**

- 主：Newsletter（深度 + 付费）
- 辅：Twitter / LinkedIn（引流）
- 适用：商业洞察 / 投资 / 行业分析

**策略 B：YouTube + Newsletter**

- 主：YouTube（受众 + 增长）
- 辅：Newsletter（深度 + 付费）
- 适用：知识 / 教程 / 评测

**策略 C：TikTok + Instagram + Newsletter**

- 主：TikTok（流量 + 增长）
- 辅：Instagram（视觉品牌）+ Newsletter（深度）
- 适用：生活方式 / 个人品牌

**策略 D：小红书 + 公众号 + 微信群**

- 主：小红书（流量 + 品牌合作）
- 辅：公众号（深度）+ 微信群（社群）
- 适用：中国市场 / 生活方式 / 教育

**策略 E：个人站 + Twitter + HackerNews**

- 主：个人站（完全控制）
- 辅：Twitter（社交）+ HN（技术社区）
- 适用：技术 / 创业 / 投资

## 平台选择的 4 个决策问题

**问题 1：你的内容深度如何？**

- **深度**：Newsletter / 个人站 / Medium
- **中等**：YouTube / 小红书
- **浅**：TikTok / Instagram

**问题 2：你的受众在哪里？**

- **全球**：YouTube / Newsletter / Twitter
- **中国**：小红书 / 公众号 / 知乎 / B 站
- **英语圈**：Newsletter / YouTube / Twitter
- **东南亚**：TikTok / Instagram

**问题 3：你的变现目标？**

- **高客单价**：Newsletter（$5-100/月）+ 1v1 咨询
- **大流量低单价**：YouTube 广告 / TikTok
- **品牌合作**：小红书 / Instagram
- **产品销售**：Newsletter + 任何流量平台

**问题 4：你的资源（时间 / 钱 / 技能）？**

- **少时间**：Newsletter（1 人就能做）
- **多时间**：YouTube / TikTok（视频制作）
- **有钱**：外包（小红书 + 多平台铺量）
- **有技能**：个人站（Ghost + 自定义）

## 实战经验

我跟踪过 50 个独立创作者（年收入 $50K+）的平台分布：

| 平台 | 主平台占比 | 辅平台占比 |
|------|----------|----------|
| Newsletter | 45% | 35% |
| YouTube | 25% | 50% |
| 个人站 | 15% | 30% |
| 小红书 | 10% | 20% |
| TikTok | 5% | 40% |
| Medium | <5% | 15% |

**Newsletter 是独立创作者主平台首选**——变现效率高、受众粘性强、长期资产。

**TikTok 是辅助平台首选**——流量大、能引流到主平台。

## 我的建议

**如果你只能选 1 个平台**：Newsletter（Substack / Beehiiv）。

**如果你想快速增长**：TikTok + Newsletter（短视频引流到订阅）。

**如果你做中国市场**：小红书 + 公众号 + 微信群。

**如果你做技术 / 商业**：个人站 + Twitter + Newsletter。
