# 05. Substack vs Beehiiv：平台深度对比

Newsletter 平台是 Newsletter 经济的"基础设施"。2026 年主流平台：Substack、Beehiiv、Kit（ConvertKit）、Ghost、Mailchimp。这一章深度对比前 2 个最重要的——Substack 和 Beehiiv。

## 平台速览

| 平台 | 定位 | 用户数 | 付费订阅分成 | 特色 |
|------|------|--------|------------|------|
| Substack | 全功能 + 网络效应 | 50 万+ 创作者 | 0%（创作者拿 100%）| 出版 + 发现 + 社交 |
| Beehiiv | 增长工具 | 30 万+ 创作者 | 0%（免费版）/ $99/月起（无分成）| 增长 / 数据 / 广告网络 |
| Kit | 营销自动化 | 100 万+ 创作者 | 0% | 自动化 + 视觉化 |
| Ghost | 自托管 + 出版 | 5 万+ | 0% | 自托管 / 完全控制 |
| Mailchimp | 通用邮件营销 | 1000 万+ | N/A | 通用邮件 |

## Substack 详细评测

### 定位

**Substack = Medium + Patreon + Twitter**。

它不只是邮件工具，更是**写作 + 出版 + 订阅 + 社区**的一站式平台。

### 核心优势

**1. 网络效应**

Substack 是头部创作者聚集地，读者订阅 Substack 是订阅"整个出版网络"——你发的内容会出现在 Substack App 里被推荐。

**2. 零抽成**

Substack 不抽成——你收多少订阅费就是多少。Stripe 处理支付（Stripe 收 2.9% + $0.30 / 笔）。

**3. 出版体验**

- 类 Medium 的阅读体验
- 自动适配移动端 / 桌面
- 嵌入式图片 / 视频 / 音频
- 评论区
- Notes（短内容，类似 Twitter）

**4. 推荐算法**

Substack App 主动推荐内容给读者，类似 Medium 的发现机制。

**5. 多平台发布**

一篇文章可以同时发：

- 邮件订阅者（邮件）
- Substack App 用户（应用内）
- 公开网页（SEO 友好）
- RSS 订阅（兼容老派读者）

### 核心劣势

**1. 品牌弱化**

所有 Substack Newsletter 都长一样（默认主题）。品牌建设受限。

**2. 自定义受限**

不能深度自定义页面 / 邮件模板 / 域名。

**3. 数据受限**

读者数据归 Substack 所有。你只能看到订阅数 / 开信率 / 点击率，不能拿到邮箱列表导出。

**4. 没有广告网络**

不像 Beehiiv 有广告网络，你得自己找赞助商。

**5. 增长工具弱**

没有 A/B 测试、referral program、advanced analytics。

### 适合谁

**适合**：

- 个人创作者 / 写作者
- 想专注内容、不想搞技术
- 看重网络效应 + 推荐
- 不需要深度自定义

**不适合**：

- 需要完全控制品牌 / 数据
- 想用广告网络变现
- 需要 referral / growth hacking 工具

### 价格

- **订阅**：免费（Substack 不抽成）
- **支付**：Stripe 处理费 2.9% + $0.30 / 笔
- **付费订阅**：Substack 不额外收费

**实战成本**：1000 付费订阅 × $5/月 = $5000/月收入 - $145 Stripe 费 = $4855/月净收入。

## Beehiiv 详细评测

### 定位

**Beehiiv = 增长 + 数据 + 广告网络的 Newsletter 平台**。

主打"帮助创作者增长 + 变现"。

### 核心优势

**1. 增长工具强**

- A/B 测试（标题 / 内容 / 发送时间）
- Referral program（订阅送积分）
- Advanced analytics（订阅来源 / 用户画像 / 转化漏斗）
- Magic Links（无密码订阅）
- Boosts（订阅者推荐其他 newsletter）

**2. 广告网络**

Beehiiv Ad Network 是 2026 年最大的 Newsletter 广告网络：

- 创作者可以卖广告位（被动收入）
- 广告主可以投 newsletter（精准受众）

**实战数据**：

- 10000 订阅 newsletter 月广告收入 $500-2000
- 50000 订阅 newsletter 月广告收入 $5000-20000

**3. 数据自主**

订阅者数据归你所有。可以导出 CSV / 同步到 CRM / 营销自动化。

**4. 自定义强**

- 自定义域名
- 自定义邮件模板
- 多个 newsletter 站点
- 嵌入式表单

**5. SEO 友好**

公开网页 SEO 优化比 Substack 好。

### 核心劣势

**1. 没有 Substack 那种 App 发现**

没有 Substack App / Notes，缺网络效应。

**2. 没有评论 / 社交**

读者之间没有互动机制。

**3. 免费版功能限制**

免费版：

- 最多 2500 订阅
- Beehiiv 品牌保留
- 基础功能

要解锁完整功能需要付费。

**4. 学习曲线**

比 Substack 复杂，新手上手要 1-2 周。

### 价格

| 计划 | 月费 | 订阅上限 | 关键功能 |
|------|------|---------|---------|
| Launch（免费）| $0 | 2500 | 基础 |
| Scale | $99/月 | 10000 | A/B 测试 / Referral |
| Max | $249/月 | 100000 | 高级分析 / 自定义 |
| Enterprise | 协商 | 无上限 | 自定义集成 |

**实战成本**：Scale 计划 $99/月 + Stripe 费 = $200-300/月运营成本。

### 适合谁

**适合**：

- 增长导向的创作者
- 想用广告变现
- 需要 referral / A/B testing
- 需要数据自主

**不适合**：

- 纯写作者（Substack 体验更好）
- 不需要广告（Substack 不收费）

## 怎么选

### 选 Substack 如果：

- 你是纯写作者，关注内容本身
- 你想被推荐（用 Substack App 发现机制）
- 你不想付平台费
- 你不需要深度数据分析

### 选 Beehiiv 如果：

- 你想规模化增长（referral / A/B testing）
- 你想用广告变现（Ad Network）
- 你需要数据自主
- 你有团队在运营

### 选 Kit 如果：

- 你已经有其他业务，想用 Newsletter 做营销
- 你需要复杂的自动化流程

### 选 Ghost 如果：

- 你想完全自托管
- 你有技术背景
- 你想完全控制品牌 / 数据

## 实战建议

### 起步阶段（< 1000 订阅）

**推荐 Substack**。零成本、零抽成、网络效应好。

**理由**：起步阶段重点是**写好内容 + 积累订阅**，不是优化工具。Substack 让你专注内容。

### 增长阶段（1000-10000 订阅）

**Substack + Beehiiv 双开**。

**理由**：用 Beehiiv 跑 referral / A/B testing 实验，用 Substack 维持核心订阅。等验证了有效策略再全量迁移。

### 规模化阶段（10000+ 订阅）

**迁移到 Beehiiv**（如果重视增长 / 广告变现）或**继续 Substack**（如果重视内容 / 推荐）。

**理由**：Substack 在规模化阶段分成优势消失（一直 0%），Beehiiv 的增长工具价值凸显。

## 真实案例对比

**创作者 A：技术 Newsletter，主打深度分析**

- 选 Substack，3 年做到 50000 订阅
- 月收入 $30000（付费 $25000 + 赞助 $5000）
- 关键成功因素：内容质量 + Substack 推荐

**创作者 B：AI Newsletter，主打资讯聚合**

- 选 Beehiiv，2 年做到 30000 订阅
- 月收入 $15000（付费 $8000 + 广告 $5000 + 产品 $2000）
- 关键成功因素：Beehiiv referral + A/B testing + 广告网络

**两种路径都成功，关键是匹配自己的内容形态**。

## 我自己的选择

我目前用 **Beehiiv**，理由：

1. 需要 A/B testing 优化标题
2. 重视 referral program
3. 想用广告网络变现
4. 需要数据自主（导出分析用户）

**如果你刚开始做**，先从 Substack 开始——零成本、专注内容。等订阅过 5000 再考虑迁移到 Beehiiv。
