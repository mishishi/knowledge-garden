# 关于这个站

我是 mishishi，2026 年开始做这个站。

起因很私人：2024 到 2025 两年里我读了太多 LLM / Agent / Harness 相关的 paper、博客、推特、播客、源码。Notion 里堆了 200 多篇 markdown，但说实话——之后再也没回看过。检索成本太高，主题也散。

知识不种下去不会自己长。所以我开了这个站，把笔记按主题整理成 5 个系列，每章 800 到 1500 字，写得能让自己一年后还读得懂。如果顺便对你也有用，那是 bonus。

## 这 5 个系列怎么选出来的

不是从市场调研来的，是我过去两年真的踩过 / 关心过的方向：

- **Multi-Agent in Practice** — 工作里写过两个 production agent，踩过 supervisor 单点 / state 共享 / cost 失控这些坑
- **LLM Prompt 实战** — 给团队培训过 3 次 prompt engineering，每次都有新发现
- **CrewAI 入门到实战** — 评估过这个框架，最后没用，但流程值得讲
- **RAG 实战** — 第一份正经工作就是搭 RAG pipeline，踩过 embedding 模型选错、chunking 拍脑袋这些坑
- **Harness Engineering** — 2026 年最关心的，写给需要自己造 / 选 harness 的人

跨 series 完全独立。Multi-Agent 04 跟 RAG 07 没有依赖关系。

## 怎么读

没有"必须按顺序"的规定。每个 series 内部是从基础到进阶，跨 series 不挑。

最快的入口是首页：那里能看 5 个系列的全貌、你的进度、有没有上次没读完的章节继续读。Cmd+K 搜关键词也行。

进度存在你浏览器 localStorage，**永远不上服务器**——我看不到你读了多少，这是设计，不是 bug。刷新页面、换电脑会重置是已知 trade-off。
