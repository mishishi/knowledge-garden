# 09. 账号 / 订阅 / 配额：15 款产品的选型决策树

15 款产品怎么订阅、怎么计费、个人 / 团队 / 企业怎么选——这一章给最实际的"我该选哪款"决策。

## 订阅模式分类

15 款产品的订阅模式分 4 种：

### 模式 1：完全免费（个人向）

- **TRAE Work 个人版**（中国版完全免费）
- **豆包任务模式**（C 端免费 + 增值）
- **QoderWork 限时**（Qwen 3.7 Max 15 天免费 + 后续调整）
- **CodeGeeX**（完全免费 + 开源）
- **钉钉 AI 个人版**（基础功能免费）
- **Marvis 每日 1000 万 Token**（个人免费）

**适合**：个人开发者、轻度用户、学生。

### 模式 2：免费 + 付费（分层订阅）

- **通义灵码**（基础免费 + 企业版付费）
- **Qoder CN**（基础免费 + Pro+ 付费）
- **CodeArts**（基础免费 + 企业版付费）
- **TRAE Work 国际版**（订阅制）
- **Marvis Pro**（订阅制）
- **Kimi Work**（基础免费 + Pro 订阅）

**适合**：独立开发者、小团队。

### 模式 3：订阅制（B 端为主）

- **WorkBuddy 企业版**（SaaS 公有云 + VPC 独占 + 私有化部署）
- **CodeBuddy**（Plugin + IDE + CLI 三形态 + Admin 后台）
- **Qoder CN 企业版**（按席位 + 模型用量）

**适合**：企业、ToB 团队。

### 模式 4：私有化部署（政企）

- **CodeGeeX**（完全开源 + 自部署）
- **文心快码 Comate 4.0**（私有化部署 + 安全扫描）

- **CodeArts 华为云原生**（国产化合规）

**适合**：政企、金融、军工、医疗。

## 4 个订阅常见问题

### Q1：可以"试"吗？

**15 款里 12 款提供免费试用**：

- TRAE Work：个人完全免费
- QoderWork：15 天免费
- WorkBuddy：内测免费
- Marvis：每天 1000 万 Token 免费
- Kimi Work：基础版免费
- Qoder CN：基础功能免费
- CodeGeeX：完全免费
- 等等

**建议**：先用免费版，**用 2-4 周**再决定付费。

### Q2：可以混搭吗？

**完全可以**——**15 款不互斥**。

**实战组合**：

- 主力：TRAE Work（免费 + 中文好）
- 长任务：Kimi Work Pro（¥99）
- 备份：CodeGeeX（免费 + 开源）
- 偶尔：Marvis（免费 + 文档处理）

### Q3：可以随时退订吗？

**15 款都支持月付 / 年付**——**退订灵活**。

**建议**：先月付，**3-6 个月稳定了再年付**。

### Q4：怎么避免被"锁定"？

**3 个避免锁定技巧**：

1. **配置文件用标准格式**（TOML / JSON）——容易迁移
2. **Skills 写在项目里**（`.agents/skills/`）——不绑定单平台
3. **数据不只在 agent 里**——**关键数据在 GitHub / 飞书 / 自建 DB**

## 我的判断

**短期（3-6 个月）**：

- 15 款里 3-5 款确立头部位置
- 其他被合并 / 关停
- **独立开发者要持续关注动态**

**中期（6-12 个月）**：

- **价格战结束**——头部 2-3 家稳定
- 独立开发者的**最优组合固定**（主力 + 辅助 + API）
- **订阅模式成熟**（个人 / 团队 / 企业三档）

**长期（12+ 个月）**：

- **桌面 Agent 成为基础设施**——像 VS Code 一样
- 订阅价格稳定，**不再"补贴换用户"**
- 独立开发者的**核心成本 = API + 订阅 = ¥100-300 / 月**
