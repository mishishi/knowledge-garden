## 为什么是这个时点

**三个驱动力同时发生**：

**驱动 1：OpenAI Codex 6/3 整合 ChatGPT**

10 亿用户、500 万 WAU、6 行业插件、110 skill、62 集成——**Codex 不再是"程序员工具"**。这事国内厂商必须跟。

**驱动 2：模型成本一年降 80%**

2025 年 GPT-5 到 2026 年 GPT-5.5，**同质量输出 token 数降到 1/3**。桌面 Agent 跑得起复杂的任务循环 + 多 Agent 编排了。

**驱动 3：企业 AI Agent 落地拐点**

腾讯云副总裁刘毅提的："**2026 下半年，企业级 AI 智能体将迎来集中落地的爆发期**"。刘毅给的依据是模型成本降 80% + Agent 可托管 + 用户习惯被教育——**三个条件同时满足**。

## 6 款核心产品 30 秒总览

### 1. TRAE Work（字节）

**核心**：从 TRAE SOLO 升级。Work / Design / Code 三模式 + 手机 / 电脑 / Web 三端协同。

**优势**：中国版完全免费（个人版全功能），中文需求理解行业领先，字节内部 600 万+ 注册用户。

**劣势**：海外生态适配弱。

**代表用户**：vibe coding 个人开发者、产品经理、前端。

### 2. QoderWork（阿里）

**核心**：从 Qoder 代码 Agent 扩到日常办公场景。底层 Qwen 3.7 Max 限时免费。

**优势**：任务模式（一次跑完整链路）、专家套件、技能市场、定时任务、应用快照。**6/16 上线"意识"功能**（记忆/反思/技能进化）。

**劣势**：测试下来更像"AI 实习生"——能干活但**离"放手让它干"还有距离**（雷科技测评 7.5/8 分）。

**代表用户**：办公白领、初级开发者、ToB 项目。

### 3. WorkBuddy（腾讯云）

**核心**：腾讯系全场景入口。3 月月访问量第一，环比增速第一，**3 个月人均 Token 增长 10 倍**。

**优势**：腾讯文档 / 网盘 / 乐享三件套打通，CodeBuddy 编程 + WorkBuddy 办公 + Managed Agents 企业自建，**B 端统一入口**。

**劣势**：C 端用户群不如豆包大。

**代表用户**：腾讯生态企业、微信小程序开发者、ToB 团队。

### 4. Marvis / Mavis（腾讯应用宝）

**核心**：操作系统层 AI 助手。1 主 + 5 副 Agent 架构（File / Computer / App / Browser / Search）。

**优势**：直接调系统 API（不是模拟键鼠），本地模式完全离线，每天 1000 万 Token 免费。

**劣势**：6 核 + 16GB 起步，机械硬盘不行，1 个月 21 个版本迭代快但偶有 bug。

**代表用户**：文件管理混乱的办公族、法务/行政/助理。

### 5. Kimi Work（月之暗面）

**核心**：**300 子 Agent 并行**、13 小时长任务、4000+ 工具调用、Kimi K2.6（1 万亿参数 MoE）**完全开源**。

**优势**：Agent Swarm 集群协作是赛道首创，**5 万行有效代码 92% 由 AI 自主生成**（Kimi 自己开发自己）。

**劣势**：300 Agent 全开风扇起飞（CPU 占用 30%+），浏览器操作偶有"点歪"，Token 消耗大。

**代表用户**：重度开发者、研究人员、复杂任务派。

### 6. 豆包任务模式（字节）

**核心**：C 端用户群最大的 AI Agent 入口，6/24 豆包专业版聚焦办公。

**优势**：用户基数大、字节内部协同（豆包 + 中兴 AI 手机）。

**劣势**：最近受"豆包付费消息"争议影响。

**代表用户**：C 端用户、移动端办公、字节生态。

## 4 个共同特征

15 款产品看似眼花缭乱，**实际有 4 个共同模式**——后文 07 章专门拆。

**模式 1：Code Agent → Desktop Agent**

所有产品都从"代码生成"扩到"通用工作"。QoderWork、WorkBuddy、Kimi Work、TRAE Work 都从代码 Agent 起步，然后**扩到写文档、做 PPT、整理文件、查数据**。**Code 不是终点，是入口**。

**模式 2：多 Agent 架构**

15 款里 12 款明确支持多 Agent 编排。QoderWork 的"专家套件"、WorkBuddy 的"专家 / 助理 / 团队"、Kimi Work 的"Agent Swarm"、TRAE Work 的"主-子 Agent"——**单 Agent 不够用是共识**。

**模式 3：Skill 生态**

15 款里 11 款做 Skill 市场。WorkBuddy SkillHub 收录 7 万+ Skills，2 月下载 3000 万。**Skill = agent 时代的 App Store**。

**模式 4：定时任务**

13 款做"定时任务 / Automations"——日报、review、备份、监控。**Agent 从"调用一次"到"持续运行"**。

## 一个核心判断

**这 15 款不是 15 场战争，是 5 场战争**：

- **字节 vs 阿里 vs 腾讯 vs 月之暗面 vs 百度**（5 大主战场）
- 每个战场内部还有 2-4 款产品互打
- **赢家一定不是"产品最多"的，是"用户最多 + 留存最久"的**

2026 下半年的关键观察点：

- 哪 1-2 款能跑到 1000 万 WAU
- 哪几家能跑通付费
- 哪些被合并 / 关停

**3-6 个月后见分晓**。
