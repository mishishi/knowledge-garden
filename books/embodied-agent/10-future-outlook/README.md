# 10. 具身智能未来 3-5 年展望

具身智能是 AI 领域最有想象力的方向——把 LLM / VLM 的大脑装到机器人的身体里。这一章讲我对未来 3-5 年的判断：技术突破点、商业爆发点、风险点、入场时机。

## 我的核心判断

**判断 1：2026-2030 是具身智能的"iPhone 时刻"**

iPhone 2007 年发布，到 2012 年渗透率才到 30%。具身智能 2025-2026 进入早期采用，2030 年前后进入主流。我预计 2030 年全球家庭服务机器人保有量 5000 万台，工业机器人 1 亿台。

**判断 2：硬件先行，软件跟上**

2026-2028 是硬件爆发期——人形机器人量产、价格下降、形态多样化。软件（VLA 模型、世界模型）2028-2030 才真正成熟。

**判断 3：中国 + 美国双极格局**

中国在硬件 + 量产领先（宇树 / 智元 / 银河 / 星动纪元）。美国在软件 + 大脑领先（OpenAI / Google / Tesla / Figure / Physical Intelligence）。两者会长期共存，中国赢硬件市场、美国赢软件生态。

## 技术突破点预测

### 2026-2027：VLA 模型收敛

**突破点**：单一 VLA 模型能处理 100+ 种任务。

代表：π0 / OpenVLA / RT-X 等基础模型达到 "通用机器人基础模型" 门槛。

**标志性事件**：Figure / 1X 公开宣布 "我们的机器人能做任何家庭家务"（虽然成功率低，但覆盖广）。

### 2027-2028：世界模型成熟

**突破点**：世界模型能预测未来 30 秒物理状态，精度 95%+。

应用：机器人能在"脑子里"模拟整个任务流程，提前发现错误。

**标志性事件**：机器人能完成 "做一道菜" 这种 30 分钟长 horizon 任务，成功率 70%+。

### 2028-2030：持续学习突破

**突破点**：机器人在真实家庭运行 1 个月后，能针对该家庭做个性化优化。

代表：基础模型 + 个性化 fine-tune。

**标志性事件**：家庭机器人用户月留存率从 60% 升到 85%+（说明用户真用起来了）。

### 2030+：通用人工智能（AGI）+ 物理世界

**突破点**：AGI 级别的 embodied agent，能处理任何物理世界任务。

**标志性事件**：家庭机器人能独立照顾独居老人（做饭 / 提醒 / 陪伴 / 紧急呼叫），责任由机器人承担。

## 商业爆发点预测

### 工业（已爆发，2024-2027）

- 工厂搬运 / 装配
- 仓储物流
- 电力 / 矿山 / 核电巡检
- 农业采摘

市场：2026 年 $200 亿，2030 年 $1500 亿。

### 商用服务（2027-2030）

- 餐饮服务（端菜 / 收桌）
- 酒店服务（送物 / 引路）
- 医疗辅助（送药 / 简单护理）
- 零售导购

市场：2028 年 $50 亿，2030 年 $500 亿。

### 家庭消费（2028-2035）

- 家务（扫地 / 做饭 / 整理）
- 老人 / 小孩看护
- 安防巡逻
- 教育陪伴

市场：2030 年 $100 亿，2035 年 $5000 亿。

### 特种作业（持续增长）

- 太空作业（NASA / SpaceX）
- 深海作业
- 灾难救援
- 军事

市场：长期稳定，2030 年 $100 亿。

## 4 个最值得关注的公司

**Tesla**（美国）

- 优势：FSD 数据积累 + Optimus 量产能力 + 自研芯片
- 风险：FSD 团队转向 Optimus，自动驾驶节奏可能放缓
- 我的判断：5 年内最有可能跑出"通用家庭机器人"

**Figure AI**（美国）

- 优势：OpenAI 深度合作 + 顶级团队 + BMW 工厂实测
- 风险：估值过高（$26 亿）
- 我的判断：3 年内工业落地最快

**宇树 Unitree**（中国）

- 优势：硬件成本控制 + 量产能力 + 性价比
- 风险：AI 大脑能力弱于美国
- 我的判断：中国本土市场最大赢家

**Physical Intelligence**（美国）

- 优势：π0 基础模型 + 顶级研究团队
- 风险：纯软件公司，硬件依赖第三方
- 我的判断：5 年内机器人"操作系统"层最大赢家

## 5 大风险点

### 风险 1：技术突破延迟

**最可能的延迟点**：VLA 模型泛化能力、世界模型预测精度。

**如果延迟 2-3 年**：2030 年具身智能不会进入家庭，停留在工业。

**判断概率**：40%。VLA 模型虽然进步快，但通用性突破还需要 2-3 年。

### 风险 2：安全事故

**可能场景**：家庭机器人撞伤小孩、夹手、误操作家电。

**影响**：监管收紧、公众信心崩塌、行业倒退 2-3 年。

**判断概率**：60%（必然会发生一些事故，关键是应对）。

### 风险 3：成本下降慢

**当前成本**：5-20 万人民币。

**目标成本**：1-3 万人民币（消费级）。

**下降速度判断**：硬件成本每年降 20-30%，软件成本接近零。2030 年达消费级目标。

**风险**：如果硬件成本下降慢于预期，2030 年家庭普及推迟到 2035 年。

### 风险 4：监管 / 伦理

**可能监管**：

- 美国：FDA 监管医疗机器人、NHTSA 监管自动驾驶
- 中国：网信办 / 工信部联合监管
- 欧盟：AI Act 2024 已生效，2026 进入执法

**影响**：合规成本上升、创新速度放缓。

**判断**：监管会跟随技术发展，不会过度抑制。

### 风险 5：经济周期

**可能场景**：全球经济衰退 → 消费降级 → 家庭机器人推迟购买。

**判断**：工业机器人抗周期（工厂需要降本增效），家庭机器人顺周期（消费品）。

## 我个人的入场建议

### 如果你是技术人 / 研究者

**2026-2027 入场最佳**。这是早期采用者阶段，类似 2008 年做移动互联网。

**学什么**：

- VLA 模型（π0 / OpenVLA / RT-X）
- 世界模型（DreamerV3 / GAIA-1）
- 仿真训练（Isaac Lab / Genesis）
- 机器人硬件基础（ROS 2）

**怎么学**：

- 入门：LeRobot（HuggingFace 开源，$300 起）
- 进阶：Franka + Isaac Lab（实验室级别）
- 高级：自购硬件 + 自建数据集

### 如果你是创业者

**2026-2028 找切入点**。这个阶段竞争还没饱和，找垂直场景。

**5 个值得做的方向**：

- **垂直行业机器人**（医疗 / 农业 / 餐饮）
- **机器人软件栈**（中间件 / VLA 模型 / 仿真平台）
- **机器人数据**（采集 / 标注 / 合成数据）
- **机器人配件**（灵巧手 / 传感器 / 电池）
- **机器人 + AI 服务**（远程操作 / 数据分析）

### 如果你是投资人

**2026-2027 布局**。具身智能是 10 年大趋势，越早入场越便宜。

**关注**：

- 团队（创始人 + 核心研究员）
- 技术路线（VLA + 世界模型 + sim-to-real）
- 商业落地（工业先 / 家庭后）
- 估值（合理估值，避开泡沫）

**避开**：

- 纯硬件公司（容易被中国价格战卷死）
- 纯应用公司（缺技术壁垒）
- 估值过高的明星公司（Figure / 1X 已经贵了）

### 如果你是普通消费者

**2027-2028 再买**。2026 年的家庭机器人能力有限 + 价格贵 + 维护麻烦。

**适合现在买的场景**：

- 工业 / 仓储（已有成熟产品）
- 安防 / 巡检（已可用）
- 教育 / 研究（可以接受早期不完美）

**不适合现在买的场景**：

- 期待"机器人管家"——离这水平差 5-10 年
- 家庭日常家务——能力还不够
- 老人看护——安全风险大

## 3 个长期信念

**信念 1：具身智能是 AGI 的必经之路**

纯文本 / 图像的 AGI 不可能真正理解世界。必须和物理世界交互，才能有"常识"。

**信念 2：硬件 + 软件融合的公司胜出**

纯硬件公司没大脑、纯软件公司没身体。最强的是 Tesla / Figure 这种"硬件 + 大脑"都做的公司。

**信念 3：未来 10 年最大的独角兽之一来自具身智能**

iPhone 时代诞生了 Uber / Airbnb / 美团 / 字节。具身智能时代也会诞生同等量级的公司——可能是 Figure / 1X / 宇树 / 智元之一，也可能是今天还没诞生的新公司。
